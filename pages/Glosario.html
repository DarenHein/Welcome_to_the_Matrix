Glosario Hacker: El Lenguaje del Subsuelo Digital

Todo hacker, ético o no, necesita hablar el idioma. No me refiero al inglés o al binario, sino al vocabulario que define las herramientas, técnicas y filosofías del mundillo. Este glosario no es solo una lista: es tu diccionario de batalla, tu primer manual de combate.

Conocer estos términos te permite entender reportes, navegar documentación, leer entre líneas en foros o charlas de conferencias. Pero más importante aún, te permite pensar como un hacker. Porque cada palabra aquí encierra una idea, una historia y una posibilidad.

Así que léelo, guárdalo, compártelo, y cada tanto, vuélvelo a visitar. Porque en este juego, el conocimiento es poder, y las palabras... también pueden ser armas.

A

  • A - Adware: Software que muestra publicidad no deseada.
  • A - APT (Advanced Persistent Threat): Amenaza persistente y avanzada a sistemas.
  • A - ARP Spoofing: Suplantación de direcciones en red local.
  • A - Autenticación multifactor: Verificación con múltiples métodos de acceso.
  • A - Análisis forense digital: Investigación de incidentes en sistemas digitales.

B

  • B - Backdoor: Acceso oculto al sistema comprometido.
  • B - Brute Force Attack: Adivina contraseñas por prueba y error.
  • B - Botnet: Red de dispositivos controlados remotamente.
  • B - Buffer Overflow: Exceso de datos sobreescribiendo memoria.
  • B - Bypass de autenticación: Saltarse el control de acceso autorizado.

C

  • C - Cifrado: Protección de datos mediante codificación secreta.
  • C - Cross-Site Scripting (XSS): Inyección de scripts en páginas web.
  • C - CSRF: Ataques que fuerzan acciones en sitios confiables.
  • C - Criptografía: Ciencia para ocultar información sensible.
  • C - Cookie Hijacking: Robo de sesión usando cookies robadas.

D

  • D - DDoS: Saturación de servicios con tráfico masivo.
  • D - DNS Spoofing: Falsificación de respuesta del servidor DNS.
  • D - Data Breach: Exposición no autorizada de información confidencial.
  • D - Dump de credenciales: Extracción masiva de datos de acceso.
  • D - Dark Web: Parte oculta y no indexada de internet.

E

  • E - Exploit: Código que aprovecha una vulnerabilidad.
  • E - Enumeración: Recolección de información de objetivos.
  • E - Eavesdropping: Escucha pasiva de comunicaciones digitales.
  • E - Escalada de privilegios: Obtener más acceso del permitido.
  • E - Ethical Hacking: Hacking legal con fines de protección.

F

  • F - Firewall: Filtra tráfico permitiendo o bloqueando conexiones.
  • F - Fuzzing: Prueba de entradas inesperadas para errores.
  • F - Forense digital: Análisis de evidencias electrónicas legales.
  • F - Force Browsing: Acceso forzado a recursos ocultos.
  • F - Firmware Hacking: Modificar software embebido en dispositivos.

G

  • G - Grey Hat: Hacker entre lo legal e ilegal.
  • G - Google Dorking: Búsqueda avanzada para encontrar información sensible.
  • G - Gusano: Malware que se propaga automáticamente.
  • G - GPS Spoofing: Falsificación de datos de ubicación GPS.
  • G - Ghidra: Herramienta de análisis de ingeniería inversa.

H

  • H - Honeypot: Sistema trampa para atraer atacantes.
  • H - Hash: Representación única de datos cifrados.
  • H - Hacker Ético: Profesional que prueba seguridad legalmente.
  • H - Hacktivismo: Hacking con fines políticos o sociales.
  • H - HTTP Smuggling: Manipulación de encabezados HTTP intermedios.

I

  • I - Ingeniería social: Manipular humanos para obtener información.
  • I - IDS (Intrusion Detection System): Sistema que detecta intrusos automáticamente.
  • I - Injection: Ingreso de código malicioso en entradas.
  • I - IP Spoofing: Falsificación de direcciones IP originales.
  • I - IoT Hacking: Explotación de dispositivos conectados a red.

J

  • J - Jailbreak: Eliminar restricciones impuestas por fabricantes.
  • J - JSON Hijacking: Robo de datos en respuestas JSON.
  • J - Jamming: Interferencia para bloquear señales inalámbricas.
  • J - JavaScript Injection: Inserción de scripts en formularios web.
  • J - JWT (JSON Web Token): Formato seguro para transmitir credenciales.

K

  • K - Keylogger: Software que registra pulsaciones del teclado.
  • K - Kerberos: Protocolo de autenticación basado en tickets.
  • K - Kali Linux: Distribución especializada para pruebas de penetración.
  • K - Kismet: Herramienta para auditoría de redes inalámbricas.
  • K - K-Anonymity: Método para anonimizar datos personales sensibles.

L

  • L - LFI (Local File Inclusion): Leer archivos internos del servidor web.
  • L - Logging: Registro de eventos para auditoría posterior.
  • L - LDAP Injection: Inyección de código en consultas LDAP.
  • L - Logic Bomb: Código que se activa bajo condiciones específicas.
  • L - Lateral Movement: Movimiento dentro de red comprometida.

M

  • M - Man-in-the-Middle: Intercepta comunicación entre dos partes.
  • M - Malware: Software malicioso con intención dañina.
  • M - MFA (Multi-Factor Authentication): Seguridad que requiere múltiples métodos de autenticación.
  • M - MAC Spoofing: Suplantación de dirección física de red.
  • M - Metasploit: Herramienta para desarrollar y ejecutar exploits.

N

  • N - Nmap: Escáner de redes para descubrir servicios activos.
  • N - Null Session: Acceso no autenticado a servicios compartidos.
  • N - NAC (Network Access Control): Control de acceso a redes empresariales.
  • N - Network Sniffing: Captura y análisis de tráfico de red.
  • N - Nessus: Escáner de vulnerabilidades ampliamente usado.

O

  • O - OSINT: Recolección de datos de fuentes públicas.
  • O - Overflow: Exceso de datos que rompe memoria.
  • O - OWASP: Proyecto que documenta riesgos de seguridad web.
  • O - Open Redirect: Redirección manipulable por el usuario.
  • O - Obfuscation: Técnica para ocultar código o lógica.

P

  • P - Phishing: Engaño para obtener credenciales falsas.
  • P - Payload: Carga útil de un exploit o malware.
  • P - Proxy: Intermediario entre cliente y servidor web.
  • P - Port Scanning: Búsqueda de puertos abiertos en red.
  • P - Patch Management: Proceso de actualización de vulnerabilidades conocidas.

Q

  • Q - Query Injection: Inyección de código en consultas automatizadas.
  • Q - QR Code Phishing: Uso de códigos QR maliciosos para engañar.
  • Q - Quarantine: Aislamiento de archivos sospechosos o infectados.
  • Q - Quick Response Attack: Ataques rápidos contra objetivos vulnerables.
  • Q - Quantum Cryptography: Criptografía basada en principios cuánticos modernos.

R

  • R - Ransomware: Malware que cifra datos y exige rescate.
  • R - Rootkit: Oculta presencia de intrusos en sistemas.
  • R - RAT (Remote Access Trojan): Control remoto encubierto de sistemas infectados.
  • R - Replay Attack: Repetición de datos capturados previamente.
  • R - Rainbow Table: Tabla para romper hashes con rapidez.

S

  • S - SQL Injection: Manipulación de bases de datos vía formularios.
  • S - Sniffing: Escucha de paquetes en redes ajenas.
  • S - Spoofing: Suplantación de identidad en red o sistema.
  • S - Spyware: Programa espía que recopila información secreta.
  • S - Social Engineering: Manipulación psicológica para obtener acceso.

T

  • T - Troyano: Malware disfrazado de aplicación legítima.
  • T - TLS (Transport Layer Security): Protocolo para cifrado de conexiones seguras.
  • T - Token Hijacking: Robo de token de sesión autenticada.
  • T - Threat Hunting: Búsqueda activa de amenazas internas.
  • T - Two-Factor Authentication: Autenticación con dos niveles de verificación.

U

  • U - UDP Flood: Inundación de paquetes que colapsan redes.
  • U - URL Obfuscation: Ocultamiento de URLs reales con engaños.
  • U - User Enumeration: Descubrimiento de usuarios válidos del sistema.
  • U - UAC Bypass: Saltarse control de cuentas de usuario.
  • U - Update Poisoning: Manipulación de actualizaciones de software.

V

  • V - Vulnerabilidad: Debilidad que puede ser explotada.
  • V - Virus: Malware que se propaga insertándose en archivos.
  • V - VPN (Virtual Private Network): Crea conexión segura sobre internet pública.
  • V - Vishing: Phishing por llamadas telefónicas manipuladas.
  • V - Virtual Patch: Solución temporal a una vulnerabilidad.

W

  • W - Worm: Malware que se propaga sin intervención humana.
  • W - WiFi Cracking: Romper claves de redes inalámbricas protegidas.
  • W - Web Shell: Acceso remoto mediante scripts en servidores web.
  • W - Watering Hole Attack: Infectar sitios frecuentados por víctimas.
  • W - Whaling: Phishing dirigido a altos ejecutivos o CEOs.

X

  • X - XSS (Cross Site Scripting): Inyección de scripts en sitios web.
  • X - XML Injection: Manipulación de datos estructurados en XML.
  • X - XSRF (Cross Site Request Forgery): Ataque que obliga a ejecutar acciones sin consentimiento.
  • X - X509 Certificate: Certificado digital usado en HTTPS.
  • X - XPath Injection: Inyección para manipular consultas en XML.

Y

  • Y - YAML Injection: Manipulación de estructuras de datos YAML.
  • Y - YARA Rules: Reglas para identificar patrones maliciosos en archivos.
  • Y - Y2K Bug: Fallo relacionado con el cambio de milenio.
  • Y - Yellow Team: Colaboración entre Blue y Red Teams.
  • Y - YouTube Hacking: Técnica de aprendizaje desde tutoriales disponibles.

Z

  • Z - Zero-Day: Vulnerabilidad aún no conocida ni parchada.
  • Z - Zombie: Dispositivo comprometido en una botnet.
  • Z - ZAP (OWASP): Herramienta para pruebas de seguridad web.
  • Z - ZIP Bomb: Archivo comprimido que explota en tamaño.
  • Z - Zscaler: Plataforma de seguridad basada en la nube.
