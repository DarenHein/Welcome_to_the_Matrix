🧪 Introducción

¿Buscas un diseño oscuro, minimalista y con vibra hacker para tu web o blog? Aquí te presento mi archivo CSS estilo Matrix, inspirado en terminales de sistema y hecho con amor al software libre.

📄 ¿Qué contiene?

Este CSS personaliza colores, tipografías y estructuras visuales para crear un efecto tipo consola con texto verde neón sobre fondo negro. Ideal para portafolios, blogs o presentaciones técnicas.

💾 Descargar

Puedes descargar el archivo aquí:

⬇️ Descargar estilos-matrix.css

⚙️ ¿Cómo se usa?

Simplemente enlaza este archivo dentro del <head> de tu HTML: 

<link rel="stylesheet" href="css/estilos-matrix.css">
